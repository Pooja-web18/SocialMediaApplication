# Social Media Application

A social media platform built with **Java**, **Spring Boot**, and **MySQL**. This application supports user registration, profile management, content sharing, and interaction through posts, comments, and real-time notifications.

## Features
- **User Registration & Login**: Secure user authentication with JWT and profile management.
- **Post Creation**: Users can create posts and share them with their followers.
- **Commenting & Liking**: Interact with posts by liking and commenting.
- **Real-time Notifications**: Real-time notifications about post activities, comments, and likes.
- **Activity Feed**: Users can see an activity feed with the latest posts from their followers.

## Technologies Used
- **Backend**: Java, Spring Boot
- **Database**: MySQL
- **Authentication**: JWT (JSON Web Token)
- **Real-Time**: WebSockets or Spring Integration for real-time notifications
- **APIs**: RESTful APIs

## Prerequisites
- Java 11 or higher
- MySQL server
- Maven (for dependency management)
